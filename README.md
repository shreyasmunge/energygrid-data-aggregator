# energygrid-data-aggregator
Fetches telemetry data from solar inverters with rate limiting


how to use?

1. colne repo to you pc
2. go to (cd energygrid-data-aggregator/python-client) and create virtual env using (python3 -m venv venv) and activate it
3. install all req by using (pip install -r requirements.txt)
4. create a .evn file in python-client folder
5. in .evn file add API_TOKEN=theToken
6. run python main.py on seperate terminal(other than npm terminal)

Before this setup mock-api:
1.copy package.json and server.js file from assignment file
2. install npm and express 
3. use new terminal to run npm start
